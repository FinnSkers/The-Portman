## ğŸ› ï¸ BACKEND FEATURES & CAPABILITIES (as of June 8, 2025)
1. CV Upload & Parsing
Upload a CV file (/api/v1/cv/upload/): Accepts PDF, DOCX, or TXT files, saves them to the server.
Parse a CV file with AI (/api/v1/cv/parse/): Extracts text and analyzes the CV using AI.
2. User Management
Register, login, and manage users (/api/v1/users/): User registration, authentication, and profile management.
JWT-based authentication: Secure endpoints with token-based access.
3. Portfolio Generation
Generate portfolios (/api/v1/portfolio/): Create interactive portfolios from CV data.
Portfolio templates: Multiple templates for different professions/styles.
4. Advanced RAG (Retrieval-Augmented Generation) Analysis
Comprehensive CV analysis (/api/v1/cv/rag/analyze/): Professional benchmarking, industry insights, improvement suggestions, and match scores.
Store and compare CV embeddings: For future professional comparisons.
5. ATS-Friendly Resume Generation
Generate ATS-optimized resumes (/api/v1/ats/): Create resumes that are optimized for Applicant Tracking Systems.
Multiple ATS templates: Choose from clean, minimal, professional, modern, etc.
6. Analytics
Get analytics and metrics (/api/v1/analytics/): Usage stats, template popularity, performance, user engagement, recent activity, and geographic data.
7. Logs
Log events (/api/v1/logs): Store and retrieve backend logs for auditing and debugging.
8. Health & System
Health check (/healthz): Check if the backend is running.
API root (/): Returns API status, version, and feature list.
